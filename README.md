## Introduction
A small and simple WPM measurement program written in Pascal, just for fun.

## Quickstart
```sh
$ make
$ ./bin/app
```

## Controls
| Key        | Action             |
| ---------- | ------------------ |
| ^Q         | Quit               |
| Space      | Finish a word      |
| Enter      | Pause/unpause      |

If you find any bugs, please create an issue and report them.
